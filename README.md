# tmux Quick Reference
A quick reference for tmux.

## Panes
**Shortcut**|**Action**
------------|----------
Prefix + %  | `split-window -h` split horizontally
Prefix + "  | `split-window -v` split vertically
Prefix + z  | `resize-pane -Z` zoom in on the current pane, repeat to zoom out
